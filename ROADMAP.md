# 🗺️ WebCraft AI Roadmap

This document outlines the planned features and improvements for WebCraft AI. We are constantly evolving and welcome community feedback and contributions!

## 🚀 Phase 1: Foundation (Current)
- [x] Voice-to-UI core engine.
- [x] Vision-to-UI (Screenshot analysis).
- [x] Real-time collaboration with Liveblocks.
- [x] Multi-tier LLM fallback (Featherless -> Gemini -> Ollama).
- [x] Instant deployment to Cloudflare subdomains.

## 🛠️ Phase 2: Enhanced Customization (Q3 2026)
- [ ] **Advanced Theme Engine**: Global style variables for colors, typography, and spacing.
- [ ] **Drag-and-Drop 2.0**: More intuitive component reordering and resizing.
- [ ] **Icon Library Integration**: Support for Lucide, FontAwesome, and Heroicons.
- [ ] **Dynamic Forms**: AI-generated contact forms with backend integration.
- [ ] **Dark Mode Support**: One-click toggle for dark/light mode on all generated components.

## 🧠 Phase 3: AI Intelligence & Content (Q4 2026)
- [ ] **AI Content Writer**: Generate marketing copy and blog posts directly on the canvas.
- [ ] **SEO Optimizer**: Automated meta tag generation and performance audits.
- [ ] **Interactive Prototypes**: Add basic logic/links between components via natural language.
- [ ] **More LLM Adapters**: Native support for Anthropic (Claude) and Groq.

## 📦 Phase 4: Ecosystem & Export (2027)
- [ ] **Export to Code**: Download projects as clean React, Vue, or raw HTML/CSS files.
- [ ] **Community Marketplace**: Share and discover custom component templates.
- [ ] **Vercel/Netlify Integration**: More deployment options beyond Cloudflare.
- [ ] **Mobile Preview**: Side-by-side view for mobile, tablet, and desktop responsiveness.

---

*Note: This roadmap is subject to change based on community feedback and technological advancements.*
